# BTC Position Pain Index Dashboard

Авторский индикатор: по истории Open Interest (Binance USDT-M Futures)
восстанавливается **средняя цена входа всех открытых фьючерсных позиций** —
аналог on-chain Realized Price, но для перпов. Без автоторговли,
без API-ключей — только публичные эндпоинты.

Скрипт полностью автономен: вся логика в одном файле `app.py`,
никаких зависимостей от других модулей проекта.

## Идея

```
OI вырос  -> открылись новые позиции по текущей цене (новый «транш»)
OI упал   -> позиции закрылись пропорционально (средний вход не меняется)
```

Из реконструкции траншей считаются:

| Метрика             | Что показывает                                            |
|---------------------|-----------------------------------------------------------|
| Рыночный безубыток  | Цена, где средняя открытая позиция в нуле — уровень-магнит |
| Индекс боли         | Взвешенная просадка открытых позиций от их входа, %       |
| Профиль входов      | Где сидят входы позиций и какая доля под водой            |
| Детектор капитуляции| Боль > 5% И OI за 24ч упал > 5% — больных выбрасывают     |

Чья боль важнее — определяется по знаку funding (перекошенная сторона).
Капитуляция исторически совпадает с локальными экстремумами:
когда больных выбросило, продавать (или откупать) больше некому.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

Откроется http://localhost:8501. Если Binance недоступен (сеть, гео-блокировка) —
дашборд переключается на синтетические demo-данные и явно это помечает.

## Светофор

- 🟢 green — цена у рыночного безубытка, существенной боли нет;
- 🟡 yellow — средняя позиция под водой: копится топливо для ликвидаций/сквиза;
- 🔴 red — капитуляция в процессе: боль высокая и OI резко падает.

## Допущения модели

История OI на Binance ограничена ~30 днями: позиции старше окна заякорены
на цене его начала, закрытия считаются пропорциональными. Для BTC-перпа,
где OI оборачивается за дни, это рабочая точность; уровни приблизительные,
важна динамика.

## Дисклеймер

Индекс показывает позиционный навес, а не точки входа. Не финансовый совет.
