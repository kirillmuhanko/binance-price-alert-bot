# BTC Volume Profile Dashboard

Аналитический Streamlit-дашборд: распределяет наторгованный объем **по цене**
(Binance USDT-M Futures) и показывает, где сидят сильные уровни.
Без автоторговли, без API-ключей — только публичные эндпоинты.

Скрипт полностью автономен: вся логика в одном файле `app.py`,
никаких зависимостей от других модулей проекта.

## Идея

| Понятие     | Что означает                                                      |
|-------------|-------------------------------------------------------------------|
| POC         | Цена с максимальным объемом — «магнит» и сильнейший уровень       |
| Value Area  | Диапазон, где прошло 70% объема (VAH — верх, VAL — низ)           |
| HVN / LVN   | Узлы объема: высокие тормозят цену, низкие («вакуум») — пролет    |

Профиль строится на трех окнах:

| Окно | Свечи | Роль                  |
|------|-------|-----------------------|
| 7д   | 15m   | Краткосрочные уровни  |
| 30д  | 1h    | Среднесрочные уровни  |
| 90д  | 4h    | Глобальные уровни     |

Дашборд автоматически находит ближайшие поддержку и сопротивление из
POC/VAH/VAL всех окон и предупреждает, если цена попала в «объемный вакуум».

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

Откроется http://localhost:8501. Если Binance недоступен (сеть, гео-блокировка) —
дашборд переключается на синтетические demo-данные и явно это помечает.

## Светофор

- 🟢 green — цена внутри Value Area 30д: рынок в балансе, движения ограничены;
- 🟡 yellow — цена вышла из зоны стоимости: решается принятие нового диапазона;
- 🔴 red — цена вне VA и в тонком объеме: тормозить нечему, вероятны резкие движения.

## Дисклеймер

Профиль показывает уровни интереса рынка, а не точки входа. Не финансовый совет.
