# BTC Funding Rate & Open Interest Dashboard

Аналитический Streamlit-дашборд: показывает **позиционирование** деривативного рынка BTC
через Funding Rate и Open Interest (Binance USDT-M Futures). Без автоторговли,
без API-ключей — только публичные эндпоинты.

Скрипт полностью автономен: вся логика в одном файле `app.py`,
никаких зависимостей от других модулей проекта.

## Идея

| Индикатор     | Что показывает                                              |
|---------------|-------------------------------------------------------------|
| Funding Rate  | Кто платит за плечо: перекос толпы в лонг или шорт          |
| Open Interest | Сколько денег в открытых позициях: приток/отток капитала    |

Связка «цена x OI» определяет режим рынка:

- цена ↑ + OI ↑ — рост на новых деньгах (тренд подтвержден);
- цена ↑ + OI ↓ — рост на закрытии шортов (хрупкий);
- цена ↓ + OI ↑ — наращивание шортов (давление вниз);
- цена ↓ + OI ↓ — разгрузка лонгов.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

Откроется http://localhost:8501. Если Binance недоступен (сеть, гео-блокировка) —
дашборд переключается на синтетические demo-данные и явно это помечает.

## Светофор

- 🟢 green — funding у базовой ставки (0.01%), OI без рывков;
- 🟡 yellow — перекос нарастает (funding выше ±0.03% или OI за 24ч изменился на 5%+);
- 🔴 red — сильный перекос: экстремальный funding и/или резкое движение OI — риск сквиза.

## Дисклеймер

Инструмент показывает позиционирование рынка, а не точки входа. Не финансовый совет.
